# Install NVM

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
nvm install node
source $HOME/.bashrc
nvm -version
nvm install node
node --version
npm install react@latest react-dom@latest next@latest
```
